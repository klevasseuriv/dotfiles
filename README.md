# dotfiles
Non-sensitive, non-proprietary workstation environment config


## Requirements:
- git
- vim,neovim
- python3,pip
- i3
- tilix
- vscode
- feh
- compton
- util-linux-user (chsh)
- zsh
- exa


## Install:
Note that any existing vscode extensions are removed before installing the defined extensions.

`./install`
